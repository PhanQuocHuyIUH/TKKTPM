# Bài 4: Static Website với Nginx

## Mô tả
Trang web tĩnh HTML đơn giản được phục vụ bởi Nginx, sử dụng `nginx:latest` làm base image.

## Cấu trúc file
```
bai4-nginx/
├── Dockerfile
├── index.html
└── README.md
```

## Giải thích Dockerfile
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM nginx:latest` | Sử dụng image Nginx mới nhất làm base |
| `COPY index.html /usr/share/nginx/html/index.html` | Copy file HTML vào thư mục mặc định của Nginx |
| `EXPOSE 80` | Nginx lắng nghe trên cổng 80 |
| `CMD ["nginx", "-g", "daemon off;"]` | Chạy Nginx ở foreground (cần thiết trong container) |

## Hướng dẫn chạy

### Bước 1: Build Docker Image
```bash
docker build -t hello-nginx .
```

### Bước 2: Chạy Container
```bash
docker run -d -p 8080:80 --name hello-nginx-container hello-nginx
```

### Bước 3: Kiểm tra
Mở trình duyệt tại **http://localhost:8080**

**Kết quả mong đợi:** Trang web hiển thị tiêu đề `Hello, Docker Nginx!`

### Dừng và xóa container
```bash
docker stop hello-nginx-container
docker rm hello-nginx-container
```

## Mở rộng: Thêm nhiều file tĩnh
Để phục vụ toàn bộ thư mục (HTML, CSS, JS, ảnh...), thay lệnh COPY trong Dockerfile bằng:
```dockerfile
COPY . /usr/share/nginx/html/
```

## Lưu ý
- `daemon off;` là bắt buộc trong Docker để tiến trình Nginx chạy ở foreground, nếu không container sẽ thoát ngay sau khi khởi động.
- Nginx mặc định phục vụ file `index.html` khi truy cập đường dẫn gốc `/`.
