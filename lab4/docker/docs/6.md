# Bài 6: Multi-stage Build với Node.js (TypeScript)

## Mô tả
Ứng dụng Node.js viết bằng **TypeScript**, sử dụng multi-stage build:
- **Stage 1** (`node:18`): Cài đặt toàn bộ dependencies và compile TypeScript → JavaScript
- **Stage 2** (`node:18-alpine`): Chỉ copy compiled JS, không có TypeScript hay devDependencies

## Cấu trúc file
```
bai6-multistage/
├── Dockerfile
├── package.json
├── tsconfig.json
├── src/
│   └── server.ts
└── README.md
```

## Giải thích Dockerfile

### Stage 1 — `build` (node:18)
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM node:18 AS build` | Image đầy đủ để build, đặt tên stage là `build` |
| `RUN npm install` | Cài tất cả deps kể cả devDependencies (typescript, @types/node) |
| `RUN npm run build` | Chạy `tsc`, compile `src/*.ts` → `dist/*.js` |

### Stage 2 — runner (node:18-alpine)
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM node:18-alpine` | Image Alpine nhỏ gọn (~170MB so với ~1GB của node:18) |
| `RUN npm install --omit=dev` | Chỉ cài production dependencies |
| `COPY --from=build /app/dist ./dist` | Lấy file JS đã compile từ stage build |
| `CMD ["node", "dist/server.js"]` | Chạy file đã compile |

## Hướng dẫn chạy

### Bước 1: Build Docker Image
```bash
docker build -t hello-multistage .
```

### Bước 2: Chạy Container
```bash
docker run -d -p 3000:3000 --name hello-multistage-container hello-multistage
```

### Bước 3: Kiểm tra
```bash
curl http://localhost:3000
```
**Kết quả mong đợi:** `Hello, Docker Multi-stage Build!`

### So sánh kích thước image
```bash
# Build một image không dùng multi-stage để so sánh
docker images hello-multistage
```
> Image cuối dùng `node:18-alpine` (~170MB) thay vì `node:18` (~1.1GB).

### Dừng và xóa container
```bash
docker stop hello-multistage-container
docker rm hello-multistage-container
```

## Tại sao cần Multi-stage Build?
| Không dùng | Dùng Multi-stage |
|------------|-----------------|
| Image chứa TypeScript compiler | Chỉ chứa compiled JS |
| Chứa devDependencies | Chỉ production deps |
| Image ~1.1 GB | Image ~170 MB |
| Source code TS bị expose | Chỉ có JS đã compile |
