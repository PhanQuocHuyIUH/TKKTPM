# Bài 10: PHP với Apache trong Docker

## Mô tả
Ứng dụng PHP đơn giản chạy trên Apache web server sử dụng `php:8.2-apache` làm base image. Bài này cũng minh họa cách mount source code từ máy host vào container.

## Cấu trúc file
```
bai10-php/
├── Dockerfile
├── index.php
└── README.md
```

## Giải thích Dockerfile
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM php:8.2-apache` | Image tích hợp PHP 8.2 + Apache, không cần cài thêm |
| `COPY index.php /var/www/html/index.php` | Copy file PHP vào document root của Apache |
| `EXPOSE 80` | Apache lắng nghe trên cổng 80 |
| `CMD ["apache2-foreground"]` | Chạy Apache ở foreground (bắt buộc trong container) |

## Hướng dẫn chạy

### Phương án 1: Build image và copy file (Dockerfile)
```bash
docker build -t hello-php .
docker run -d -p 8080:80 --name php-container hello-php
```

### Phương án 2: Mount source code từ máy host (không cần build lại)
```bash
# Windows PowerShell
docker run -d -p 8080:80 --name php-container -v ${PWD}:/var/www/html php:8.2-apache

# Windows CMD
docker run -d -p 8080:80 --name php-container -v %cd%:/var/www/html php:8.2-apache

# Linux/macOS
docker run -d -p 8080:80 --name php-container -v $(pwd):/var/www/html php:8.2-apache
```
> Khi dùng volume mount, mọi thay đổi file PHP trên máy host sẽ **phản ánh ngay lập tức** trong container — không cần build lại image!

### Kiểm tra
Mở trình duyệt tại **http://localhost:8080**

**Kết quả mong đợi:**
- Tiêu đề: `Hello, Docker PHP!`
- Hiển thị: phiên bản PHP, thông tin server Apache, hostname container, thời gian server

### Xem logs Apache
```bash
docker logs php-container
```

### Dừng và xóa container
```bash
docker stop php-container
docker rm php-container
```

## Mở rộng: Cài thêm PHP extensions
Thêm vào Dockerfile trước `CMD`:
```dockerfile
RUN docker-php-ext-install pdo pdo_mysql mysqli
```

## So sánh COPY vs Volume Mount
| | `COPY` trong Dockerfile | Volume Mount (`-v`) |
|-|------------------------|---------------------|
| Khi dùng | Production | Development |
| File thay đổi | Cần build lại image | Tức thì |
| Độc lập | Image tự chứa code | Phụ thuộc máy host |
