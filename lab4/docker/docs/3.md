# Bài 3: React Docker Application

## Mô tả
Ứng dụng React được build bên trong Docker sử dụng `node:18-alpine`, sau đó file tĩnh được phục vụ qua Nginx. Đây là mô hình multi-stage build phổ biến cho các ứng dụng frontend.

## Cấu trúc file
```
bai3-react/
├── Dockerfile
├── package.json
├── public/
│   └── index.html
├── src/
│   ├── App.js
│   └── index.js
└── README.md
```

## Giải thích Dockerfile
### Stage 1 - Build (node:18-alpine)
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM node:18-alpine AS build` | Base image nhẹ Alpine cho bước build, đặt tên stage là `build` |
| `RUN npm install` | Cài đặt tất cả dependencies |
| `RUN npm run build` | Compile và bundle React app thành file tĩnh trong `/app/build` |

### Stage 2 - Serve (nginx:alpine)
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM nginx:alpine` | Image Nginx nhỏ gọn để phục vụ file tĩnh |
| `COPY --from=build /app/build ...` | Chỉ copy thư mục build từ stage trước, loại bỏ node_modules |
| `EXPOSE 80` | Nginx phục vụ trên cổng 80 |

## Hướng dẫn chạy

### Bước 1: Build Docker Image
```bash
docker build -t hello-react .
```
> Lưu ý: Bước này có thể mất vài phút do cài npm install và build React.

### Bước 2: Chạy Container
```bash
docker run -d -p 8080:80 --name hello-react-container hello-react
```
> Cổng 80 của container được map sang cổng 8080 trên host.

### Bước 3: Kiểm tra
Mở trình duyệt tại **http://localhost:8080**

**Kết quả mong đợi:** Trang web hiển thị `Hello, Docker React!`

### Dừng và xóa container
```bash
docker stop hello-react-container
docker rm hello-react-container
```

### Xóa image
```bash
docker rmi hello-react
```

## Lợi ích của multi-stage build
- Image cuối không chứa source code hay node_modules (chỉ còn ~20MB với nginx:alpine)
- Bảo mật hơn: không expose code gốc
- Image nhỏ hơn đáng kể so với dùng thẳng node:18
