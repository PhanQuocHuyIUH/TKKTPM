# Bài 2: Python Flask Docker Application

## Mô tả
Ứng dụng Flask hiển thị `Hello, Docker Flask!` chạy trên cổng 5000, sử dụng `python:3.9` làm base image.

## Cấu trúc file
```
bai2-flask/
├── Dockerfile
├── app.py
├── requirements.txt
└── README.md
```

## Giải thích Dockerfile
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM python:3.9` | Sử dụng image Python 3.9 làm base |
| `WORKDIR /app` | Thiết lập thư mục làm việc |
| `COPY requirements.txt .` | Copy file requirements trước để tận dụng cache |
| `RUN pip install --no-cache-dir -r requirements.txt` | Cài đặt dependencies, `--no-cache-dir` giữ image nhỏ hơn |
| `COPY . .` | Copy toàn bộ source code |
| `EXPOSE 5000` | Khai báo cổng Flask mặc định |
| `CMD ["python", "app.py"]` | Chạy ứng dụng Flask |

## Hướng dẫn chạy

### Bước 1: Build Docker Image
```bash
docker build -t hello-flask .
```

### Bước 2: Chạy Container
```bash
docker run -d -p 5000:5000 --name hello-flask-container hello-flask
```

### Bước 3: Kiểm tra
Mở trình duyệt tại **http://localhost:5000** hoặc dùng curl:
```bash
curl http://localhost:5000
```
**Kết quả mong đợi:** `Hello, Docker Flask!`

### Xem logs
```bash
docker logs hello-flask-container
```

### Dừng và xóa container
```bash
docker stop hello-flask-container
docker rm hello-flask-container
```

### Xóa image
```bash
docker rmi hello-flask
```
