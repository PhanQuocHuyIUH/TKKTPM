# Bài 9: Redis với Cấu Hình Tùy Chỉnh

## Mô tả
Dockerfile chạy Redis với file cấu hình `redis.conf` tùy chỉnh, cho phép kiểm soát memory, persistence, logging và nhiều tham số khác.

## Cấu trúc file
```
bai9-redis/
├── Dockerfile
├── redis.conf
└── README.md
```

## Giải thích Dockerfile
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM redis:latest` | Base image Redis mới nhất |
| `COPY redis.conf /etc/redis/redis.conf` | Copy file cấu hình vào container |
| `EXPOSE 6379` | Cổng Redis mặc định |
| `CMD ["redis-server", "/etc/redis/redis.conf"]` | Chạy Redis với file cấu hình tùy chỉnh |

## Các tham số cấu hình trong redis.conf
| Tham số | Giá trị | Ý nghĩa |
|---------|---------|---------|
| `bind 0.0.0.0` | - | Lắng nghe trên tất cả interface |
| `port 6379` | 6379 | Cổng Redis |
| `maxmemory 256mb` | 256MB | Giới hạn bộ nhớ tối đa |
| `maxmemory-policy allkeys-lru` | LRU | Xóa key ít dùng nhất khi đầy bộ nhớ |
| `save 900 1` | - | Snapshot nếu ≥1 key thay đổi trong 900 giây |
| `save 300 10` | - | Snapshot nếu ≥10 key thay đổi trong 300 giây |
| `save 60 10000` | - | Snapshot nếu ≥10000 key thay đổi trong 60 giây |
| `loglevel notice` | notice | Mức độ log |

## Hướng dẫn chạy

### Bước 1: Build Docker Image
```bash
docker build -t custom-redis .
```

### Bước 2: Chạy Container
```bash
docker run -d -p 6379:6379 --name redis-container custom-redis
```

### Bước 3: Kết nối và kiểm tra
**Dùng redis-cli bên trong container:**
```bash
docker exec -it redis-container redis-cli
```

**Các lệnh kiểm tra cơ bản:**
```redis
PING               # Kết quả: PONG
SET name "Docker"  # Lưu giá trị
GET name           # Lấy giá trị → "Docker"
CONFIG GET maxmemory  # Kiểm tra cấu hình maxmemory
INFO server        # Thông tin server
EXIT
```

### Xem logs
```bash
docker logs redis-container
```

### Dừng và xóa container
```bash
docker stop redis-container
docker rm redis-container
```

## Mở rộng: Bật xác thực (Authentication)
Thêm vào `redis.conf`:
```
requirepass your_strong_password
```
Kết nối có xác thực:
```bash
docker exec -it redis-container redis-cli -a your_strong_password
```
