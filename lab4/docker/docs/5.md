# Bài 5: Go Docker Application

## Mô tả
Ứng dụng web Go đơn giản, được build thành binary tĩnh bằng `golang:1.21-alpine` và chạy trên image `alpine:latest` rất nhỏ gọn (~10MB).

## Cấu trúc file
```
bai5-go/
├── Dockerfile
├── go.mod
├── main.go
└── README.md
```

## Giải thích Dockerfile
### Stage 1 - Build (golang:1.21-alpine)
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM golang:1.21-alpine AS build` | Image Go để compile code |
| `COPY go.mod ./` | Copy file module trước |
| `RUN go mod download` | Tải dependencies (nếu có) |
| `RUN go build -o main .` | Compile toàn bộ package thành binary `main` |

### Stage 2 - Run (alpine:latest)
| Lệnh | Ý nghĩa |
|------|---------|
| `FROM alpine:latest` | Image Alpine cực nhỏ (~5MB) chỉ để chạy binary |
| `COPY --from=build /app/main .` | Copy binary đã build từ stage 1 |
| `CMD ["./main"]` | Chạy trực tiếp binary Go |

## Hướng dẫn chạy

### Bước 1: Build Docker Image
```bash
docker build -t hello-go .
```

### Bước 2: Chạy Container
```bash
docker run -d -p 8080:8080 --name hello-go-container hello-go
```

### Bước 3: Kiểm tra
```bash
curl http://localhost:8080
```
**Kết quả mong đợi:** `Hello, Docker Go!`

### Kiểm tra kích thước image
```bash
docker images hello-go
```
> Image cuối chỉ khoảng ~10-15MB nhờ multi-stage build với alpine.

### Dừng và xóa container
```bash
docker stop hello-go-container
docker rm hello-go-container
```

## Lợi ích
- Go compile thành static binary → không cần runtime thêm
- Image cuối cực kỳ nhỏ so với golang:1.21 (~300MB)
- Thích hợp cho production deployment
